# dbg.wait

```lua
dbg.wait(timeout)
```

等待调试器进入

## 参数表

Name | Type | Description
-----|------|--------------
`timeout`|`int`| 超时秒数,默认120,即120秒

## 返回值

> *无返回值*

# dbg.stop

dbg.stop()

## 参数表

> 无参数

## 返回值
